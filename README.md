# Next.js Contact Manager – Bitcot Coding Challenge

This project is a **Contact Management Application** built using **Next.js and Tailwind CSS** as part of the Bitcot coding challenge.

The application allows users to manage contacts through a clean interface with features like searching, adding, editing, viewing, and deleting contacts.

---

## 🚀 Live Demo

https://akash-bitcot.vercel.app

---

## 📌 Features

* Fetch contacts from external JSON API
* Display contacts in a list view
* Search contacts by name or phone number
* Add new contact (stored locally in React state)
* Edit existing contact
* Delete contact
* View detailed contact information
* Modal-based UI for forms

---

## 🛠 Tech Stack

* Next.js
* React
* Tailwind CSS
* JavaScript
* Fetch API

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/Ak003x/akash-bitcot.git
```

Install dependencies

```
npm install
```

Run development server

```
npm run dev
```

Open browser

```
http://localhost:3000
```
